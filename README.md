#  Hotel VS — Hotel Management System

A modern and fully functional Hotel Management System built using **React.js**. This project helps manage hotel rooms, bookings, billing, reviews, wishlist features, and customer interactions — all in one place!

---

## 🌟 Features

* 🔐 Login & Logout system
* 👋 Welcome screen after login
* 📊 Dashboard with real-time stats
* 🛏️ Room management with availability tracking
* 💰 Room pricing display
* ⭐ Ratings & Reviews system
* 🔍 Search, Filter & Sort rooms
* 📋 Room booking with confirmation popup
* 📜 Booking history with check-out option
* 💰 Auto billing calculation based on days & room type
* ❤️ Wishlist / Favorite rooms
* 💾 Data saved using localStorage
* ☰ Hamburger menu for sidebar
* 📱 Responsive modern UI
